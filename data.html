What is WRE (weekly run effect)?

It is my attempt at capturing the training effect of the combination of walking, trail running, and road running that I in to a single metric.
Roughly, it is the distance that would have been run in a given week if all my running had been at a pace of 6min/km on flat roads.
For a detailed explanation, read on.
  • We know walking and trail running count as aerobic training that is very specific to running.
  • But while doing a lot of walking and trail running, knowing how much I ran in a given month is not straightforward!
  • We seek to define a new metric (WRE) that allows comparison of training load from month to month.
  • Define grade adjusted distance as the distance plus 1km for every 120m of elevation gain.
  • GAP (grade adjusted pace) is the pace (in min/km) corresponding to the grade adjusted distance.
  • GAP allows us to compare paces across months.
  • We would also like to account for the greater training effectiveness of faster paces.
  • Assume training effectiveness increases linearly with speed (double the speed has double the effect).
  • For example, 1km run in 3 minutes is as effective as 2km run in 6 minutes.
  • Define a RE (run effect) point as the training effect of running 1km in 6 minutes.
  • Therefore, a kilometre run in 3 minutes will receive 2 RE points, as will 2km run in 24min (at 12min/km).
  • We use this relationship to compute how many RE points were accumulated in a given month.
  • WRE (weekly run effect) is the average number of points accumulated during any given week in a given month.
  • WRE standardises the training effect of running fast, running slow, walking, and trail running.
  • WRE represents how many kilometres I would have run every week if all I did is run at a steady pace of 6min/km on flat roads.
  • We can now directly compare training load between different months by comparing their WREs.
  • Pretty useful, isn't it? :)

Notes on walking

  • "Walk km" refers to distance of purposeful walking without racewalking technique for low end aerobic stimulus and long distance training.
  • Indoor walking, short outdoor chores, and commutes are not included.
  • Walking is probably the most transferrable exercise to running, especially training for events that last 6+ hours.
  • I usually walk at a pace of around 9:00min/km on flat roads.
  • Casual social hiking is included in "Walk km" instead of "Trail run km". Slow social walking is omitted entirely.
